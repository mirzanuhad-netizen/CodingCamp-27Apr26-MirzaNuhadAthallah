# Expense & Budget Visualizer
**Coding Camp Project — Mirza Nuhad Athallah (27 Apr 2026)**

A mobile-friendly web app to track daily spending. Shows total balance, transaction history, and a visual pie chart of spending by category. All data is stored locally in the browser — no backend required.

---

## Features

### MVP
- ✅ Add transactions (name, amount, category: Food / Transport / Fun)
- ✅ Form validation — all fields required
- ✅ Scrollable transaction list with delete
- ✅ Total balance auto-updates
- ✅ Pie chart by category (Chart.js), auto-updates

### Optional Challenges (3 of 5)
- ✅ Sort transactions by amount or category
- ✅ Highlight spending over a set limit
- ✅ Dark / light mode toggle

| Layer      | Technology                        |
